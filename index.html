WSPR Matrix

A browser-based tool for turning live ClickHouse WSPR reports into visual daily activity matrices by reporter, band, UTC time, local time, and selected destination areas.

Example matrix output. Click the image to open the full-size visual.

See when paths are active

WSPR Matrix collects live ClickHouse data or loads saved collected data, then builds a 30-minute table showing observed activity by band. It can compare two selected destination areas and also show all destinations heard by the reporter.

The app runs locally in your browser. No server is required for the browser app itself.

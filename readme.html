Read Me

WSPR Matrix is a browser app for reviewing observed WSPR activity from a selected reporter across a UTC day.

Purpose

The app helps radio operators and propagation researchers see daily activity patterns by band, 30-minute UTC time slot, and selected destination area.

What The App Does

  • Imports saved WSPRnet data or collected live data, then builds a 30-minute UTC table showing observed activity by band.
  • Builds a two-path matrix using selected destination areas and endpoint radii.
  • Builds an all-destinations matrix for the same reporter and date.
  • Supports raw spots, unique beacons per bin, and cumulative unique beacons.
  • Exports CSV and supports browser print/PDF.

Scientific Boundary

The matrix shows observed activity. It does not, by itself, prove short-path or long-path propagation. Directional path claims need separate evidence.
